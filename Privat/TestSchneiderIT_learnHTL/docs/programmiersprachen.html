Programmiersprachen

Die verschiedenen Programmiersprachen, die Sie lernen können, sind: C#, HTML, CSS, SCSS, JavaScript, Bootstrap, WPF, Brainfuck.
Einige davon sind eigentlich keine Programmiersprachen.

Unterkategorien der Programmiersprachen

C#

Hier lernen Sie alles über die objektorientierte Programmiersprache C#, Variablen, Methoden, Klassen und vieles mehr.

HTML

Hier lernen Sie alles über die Beschreibungssprache HTML. Diese Sprache befindet sich aktuell in der fünften Version.

CSS

Hier lernen Sie alles über die Beschreibungssprache CSS. Diese Sprache befindet sich aktuell in der dritten Version.

JavaScript

Hier lernen Sie alles über die Programmiersprache JavaScript.

Bootstrap

Hier lernen Sie die CSS-Klassen-Sammlung Bootstrap kennen.

SASS

Hier lernen Sie alles über die Beschreibungssprache SASS. Dies ist eine verbesserte Erweiterung für CSS. Im Endeffekt soll sie das Programmieren erleichert.

Brainfuck

Hier lernen Sie alles über die Programmiersprache mit dem weltweit kleinsten Compiler, BRAINFUCK!

WPF

Hier lernen Sie alles über WPF. Das ist zwar schon ein wenig veraltet, sodass es niemand mehr nutzt, aber Sie können es trotzdem lernen.

C++

Hier lernen Sie alles über die Programmiersprache C++. Der Vorgänger von C#.

React

Hier lernen Sie alles über die Programmiersprache C++. Der Vorgänger von C#.
