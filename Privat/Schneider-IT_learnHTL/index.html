HTML
HTML ist eine Beschreibungssprache, die zur Entwicklung von Websiten dient. Sie wurde am 13. März 1989 von Tim Berners-Lee am CERN in Genf vorgeschlagen, damit sich Wissenschaftler miteinander über Ihre Forschungen verständigen können.

Heutzutage benutzt jede Website im Internet HTML, damit die verschiedenen Bilder, Texte, Grafiken oder Animationen strukturiert dargestellt werden können. HTML gibt den Inhalt aller Websiten vor.

Die Abkürzung HTML steht übrigens für Hypertext Markup Language. Auch, wenn HTML keine richtige Programmiersprache, sondern nur eine ledigliche Beschreibungssprachen, ist, finden Sie unter Programmiersprachen im Header mehr zu HTML.
CSS
Cascading Style Sheets ist neben HTML ebenfalls einer der wichtigsten Standards im gesamten Internet. Während sich HTML um die Struktuierung, den Inhalt kümmert, sorgt CSS für das Styling. CSS positioniert, rotiert, färbt, vergrößert, animiert, kontrolliert und ergänzt den Inhalt anschließend so, wie es der Programmierer will.

Hierbei sind drei verschiedene Methoden von CSS möglich: eine externe CSS-Datei, das style-Element oder das style-Attribut. Das lernen Sie beim CSS-Kurs noch detaillierter.

Genau wie HTML zählt CSS auch zu den Beschreibungssprachen. Trotzdem finden Sie unter Programmiersprachen im Header mehr zu CSS.
NWTK
Die Netzwerktechnik beschäftigt sich mit dem manchmal logischen, manchmal Hardware-technischen Aufbau aller technischer Geräte. Egal, ob WLAN, Datentransport, IP-Adressen-Zuweisung oder andere interessante Standards des Internets, NWTK kümmert sich darum.

Alle Funktionalitäten, die mit moderner Technik zu tun haben, sind standardisiert, das bedeutet, es gibt Vorschriften, die vorgeben, wie diese Technik zu funktionieren hat beziehungsweise was diese Technik alles können muss.

Klicke im Header auf Netzwerktechnik, um mehr über dieses interessante Gebiet der Technik zu erfahren.
Bootstrap
Bootstrap ist eine Sammlung von CSS-Klassen. Es erleichtert das Programmieren mit HTML enorm, wenn man eine standardisierte Website haben will. Die verschiedenen Klassen sind sehr einfach zu verstehen. Dafür kann man halt auch nicht alles damit erreichen. Wenn man spezifische Animation haben will, muss man das eigene CSS-Wissen auspacken.

Trotzdem kann man ziemlich dankbar sein, dass es Bootstrap gibt.

Bootstrap ist keine Programmiersprache und keine Beschreibungssprache. Doch Bootstrap hat viel mit HTML und CSS zu tun, weshalb Sie auch Bootstrap unter Programmiersprachen im Header finden können.
JavaScript
JavaScript regelt alles. Mit JavaScript kann man Animation machen, Inhalte in HTML hinzufügen, weglöschen oder automatisch generieren lassen, man kann 3D-Animationen in seine Website integrieren oder auch 3D-Modell zeichnen. Mit JavaScript kann man all das machen, was man mit HTML und CSS nicht erreichen kann.

Der einzige Nachteil: JavaScript ist eine schwierige Programmiersprache. Es kann gut sein, dass man als Programmierer 5 Stunden lang einen Fehler in JavaScript sucht, bis man in schließlich endlich findet, weil Javascript Fehler nicht visuell hervorhebt und kennzeichnet.

Ein unglaublich tolles Tool für HTML und für andere Programme. Mehr darüber erfahren Sie, wenn sie im Header auf Programmiersprachen klicken.
SYTB
SYTB ist die Abkürzung für Betriebssysteme und wie der Name schon sagt, geht es in diesem Fach um die verschiedensten Betriebssysteme. Egal, ob Debian, Ubuntu oder Windows, in SYTB wird Ihnen gelehrt, wie Sie selbst die verschiedensten Betriebssysteme aufsetzen können, konfigurieren können und darauf auch arbeiten können.

Sie wollen einen Server aufsetzen, wissen aber nicht wie? SYTB!
Sie wollen auf ihrem Computer Software installieren, die Malware enthalten könnte? SYTB!
Sie wollen eine Virtuelle Maschine aufsetzen, auf der Sie ins DarkNet kommen? SYTB!

SYTE
In Elektrotechnik geht es um die Lehre der elektronischen Bauteile und deren mathematische elektrotechnischen Berechnung von Strömen, Spannungen und Widerständen.

Wenn Sie wissen wollen, wie man Spannungen oder Ströme von Transistoren, einfachen Schaltungen oder Dioden berechnet, klicken Sie einfach auf Systemtechnik im Header und Sie werden direkt weitergeleitet.
(Mathematische Vorkenntnisse der komplexesten Operationen sind herzlichst erwünscht!)
INSY
INSY steht für Informationssysteme. Hierbei geht es ganz einfach um die ideale logische Speichervarianten von Daten. Überlegen Sie sich einmal: Wie speichert man Daten am besten? In einer Datenbank!
Apropos Datenbanken: Wenn Sie mehr darüber wissen wollen, klicken Sie einfach im Header auf Datenbanken und Sie werden zu zahllosen interessanten Themen weitergeleitet.

Was sind Datenbanken, Beziehungen und Tabellen? Wie speichert man Zusammenhänge zwischen verschiedenen Objekten am besten? Wie organisiert und strukturiert man den Speicher?

INSY ist das HTML von Datenbanken: Es hat Struktur.
C#
C# (Aussprache: "C Sharp" / [siːˈʃɑːp]) ist eine moderne objektorientierte Programmiersprache mit deren Hilfe man Programme entwickeln kann. Die Sprache ist eine Weiterentwicklung von C, C++ und Java und unterstützt sowohl die Entwicklung von sprachunabhängigen .NET-Komponenten als auch COM-Komponenten für den Gebrauch mit Win32-Anwendungsprogrammen.

Zum Namen C#: # (sharp) ist in der Musikwelt die Erhöhung eines Grundtones um einen Halbton. Dies könnte ein Anspielung darauf sein, dass der Name der Programmiersprache C++ der Notation für die Erhöhung des Werts einer Variable um eins entstammt. Des Weiteren kann man das Kreuz-Zeichen als Kombination von vier Plus-Zeichen betrachten, was eine Erhöhung von C++ darstellen soll.

Mithilfe von C# kann man jedes Programm, jede App entwickeln, die man will, man benötigt nur genügend C#-Kenntnisse. Klicke auf Programmiersprachen im Header, um dir weitere Informationen über C# durchzulesen.
